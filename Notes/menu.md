# 🍽️ Cookwise Pro Menu Overview

<div style="border-left: 4px solid #0078D4; padding-left: 15px; margin: 10px 0;">
  <p><strong>The menu bar is your command center – every major feature is one click away.</strong><br>
  Helpful icons and clear labels make navigation intuitive for both novice and power users.</p>
</div>

---

🔹 All Recipes
Icon: <i class="fas fa-list-alt"></i>
Show the full recipe library. Resets any filters/search so you always return to the complete collection.

📤 Share All Recipes
Icon: <i class="fas fa-share-alt"></i>
Generate a text‑based export of every recipe, with descriptions, ingredients, directions and notes. Uses the Web Share API when available, or copies the content to clipboard.

🗑️ Delete All Recipes
Icon: <i class="fas fa-trash-alt"></i> (destructive, red)
Wipe your entire database. A confirmation prompt prevents accidental loss.

🧪 Conversion Chart
Icon: <i class="fas fa-exchange-alt"></i>
Quick reference for unit conversions (cups ⇄ grams, °F ⇄ °C, etc.).

📸 Photo Album
Icon: <i class="fas fa-image"></i>
Browse all recipe images in a browsable, filterable gallery. Photos are grouped by recipe, and you can click any image to view the recipe details.

💡 Recipe Suggestions
Icon: <i class="fas fa-lightbulb"></i>
AI‑driven ideas based on pantry items, dietary preferences, or past meals.

📚 Tutorial
Icon: <i class="fas fa-graduation-cap"></i>
Step‑by‑step walkthrough covering all major features and tips to get the most from Cookwise Pro.

⌨️ Keyboard Shortcuts
Icon: <i class="fas fa-keyboard"></i>
Open a modal listing every available shortcut (e.g. Ctrl/Cmd + G for gallery, Ctrl/Cmd + Q for All Recipes, etc.).

⚙️ Settings & Utilities (not shown above)
⚙️ Settings – toggle dark mode, manage storage
🧊 Export / Import – backup or restore your data
🧹 Clear Cache – reset local storage without losing recipes

**Design notes:**

- Icons use the standard Font-Awesome classes for consistency.
- Destructive actions are colored red and carry warning text.
- Group related items (data management, navigation, help) using separators or sub-menus in the actual app UI.
- Include tooltips that mirror these descriptions for accessibility.

</div>
